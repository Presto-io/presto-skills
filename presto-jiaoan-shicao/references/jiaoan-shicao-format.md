# 实操教案模板格式参考

`jiaoan-shicao` 模板会把 Markdown 标题层级和段落映射成横向 A4 教学活动表。生成内容时应优先使用标题层级表达教案结构，不要手写普通 Markdown 表格来模拟教案表。

## 结构映射

| Markdown 输入 | PDF 中的位置 |
|---|---|
| `template: "jiaoan-shicao"` | 选择实操教案模板 |
| `## 教学活动设计` | 居中的章节标题；多个 `##` 自动另起一页 |
| `### 学习环节——学习单元` | 新建一张教学活动表，并拆成“学习环节”和“学习单元” |
| `#### 活动一` | 表格第一列“教学活动”，自动编号 |
| `##### 0.5H` | 表格最后一列“课时分配” |
| `#####` 后第一个内容块 | “学习内容”列 |
| `#####` 后第二个内容块 | “学生活动”列 |
| `#####` 后第三个内容块 | “教师活动”列 |
| `#####` 后第四个内容块 | “教学方法与手段”列 |
| `{pagebreak}` | 强制换页，并延续当前结构 |
| `{pagebreak:weak}` | 弱分页 |
| `同上` | 与上一行同列单元格合并 |

## PDF 表格列

表格固定生成六列：

| PDF 列 | 来源 |
|---|---|
| 教学活动 | `####` 标题 |
| 学习内容 | `#####` 后第一个内容块 |
| 学生活动 | `#####` 后第二个内容块 |
| 教师活动 | `#####` 后第三个内容块 |
| 教学方法与手段 | `#####` 后第四个内容块 |
| 课时分配 | `#####` 标题 |

表格第一行还会展示：

| PDF 区域 | 来源 |
|---|---|
| 学习环节 | `###` 标题分隔符前半部分 |
| 学习单元 | `###` 标题分隔符后半部分 |

## 章节标题

`##` 显示为页面上方居中的章节标题。如果文档里有多个 `##`，从第二个 `##` 开始会自动另起一页。

```markdown
## 教学活动设计——PLC 基本指令应用
```

## 学习环节和学习单元

`###` 会新建一张表。标题会被拆成两部分：

```markdown
### 认识 PLC 硬件——了解 PLC 的基本组成与接线方法
```

对应 PDF：

| PDF 表格位置 | 内容 |
|---|---|
| 学习环节 | 认识 PLC 硬件 |
| 学习单元 | 了解 PLC 的基本组成与接线方法 |

支持分隔符：`——`、`—`、`-`。推荐中文长横线 `——`。如果不写分隔符，“学习环节”显示完整标题，“学习单元”为空。

## 教学活动

`####` 进入“教学活动”列，并由模板自动编号。同一张表中的后续 `####` 会自动编号为 `2.`、`3.`。

```markdown
#### 活动一：PLC 硬件认知
```

不要为了控制编号手动写 `1.`。可以保留“活动一”作为活动名称的一部分。

## 课时分配和四个内容块

`#####` 进入“课时分配”列。同一个 `####` 下可以有多个 `#####`，每个 `#####` 生成一行具体教学安排。

```markdown
##### 0.5H

PLC 的基本组成：CPU 模块、输入模块、输出模块、电源模块。

观察实训台上的 PLC 设备，识别各模块位置及功能。

展示 PLC 实物，讲解各模块的功能与作用。

实物展示、讲练结合
```

内容块顺序：

| 顺序 | PDF 列 |
|---|---|
| 第一个内容块 | 学习内容 |
| 第二个内容块 | 学生活动 |
| 第三个内容块 | 教师活动 |
| 第四个内容块 | 教学方法与手段 |

如果少写某个内容块，对应单元格会留空。为了表格完整，建议每个 `#####` 后都写满四个内容块。

## 多行内容自动编号

在前三个内容列中，如果一个内容块内部有多行文字，模板会给每行自动加序号和分号。

```markdown
##### 1H

认识 PLC 输入模块
认识 PLC 输出模块
理解电源模块作用

观察实训台
完成模块识别记录

展示实物
点评识别结果

讲练结合
```

“教学方法与手段”列通常居中显示，不建议写成过长段落。

## 多个课时行

同一个活动下的多个 `#####` 会生成多行，“教学活动”列纵向合并。

```markdown
#### 活动一：PLC 硬件认知

##### 0.5H

PLC 的基本组成。

观察 PLC 设备。

展示 PLC 实物。

实物展示

##### 0.5H

PLC 的外部接线方法。

完成输入、输出端子接线。

示范接线步骤，巡回指导。

任务驱动
```

## 多个活动和多张表

同一张表中可写多个 `####`。同一个 `##` 章节下可写多个 `###`，每个 `###` 生成一张教学活动表。同一章节、同一分页段内的表格会共享列宽，便于视觉对齐。

## 分页

`{pagebreak}` 强制换页；如果分页发生在当前活动内部，后续 `#####` 会延续分页前的学习环节、学习单元和教学活动。

```markdown
##### 0.5H

PLC 的基本组成。

观察 PLC 设备。

展示 PLC 实物。

实物展示

{pagebreak}

##### 0.5H

PLC 的外部接线方法。

完成端子接线。

示范接线步骤。

任务驱动
```

`{pagebreak:weak}` 生成弱分页，空间不足时更倾向于换页。不确定时优先使用 `{pagebreak}`。

## 使用“同上”合并单元格

只要某个单元格内容包含 `同上`，模板会尝试把它与正上方同列单元格合并。不要在第一行使用 `同上`，因为上方没有可合并内容。

```markdown
#### 活动一：PLC 硬件认知

##### 0.5H

PLC 的基本组成。

观察 PLC 设备。

展示 PLC 实物。

实物展示

##### 0.5H

同上

完成模块识别记录。

同上

同上
```

## 常见错误

少写内容块会导致对应列留空。除非用户明确希望空列，否则补足学习内容、学生活动、教师活动、教学方法与手段。

不要把普通 Markdown 表格写进教案模板：

```markdown
| 学习内容 | 学生活动 |
|---|---|
| 内容 | 活动 |
```

这种表格不会被模板当作实操教案结构。应使用 `##`、`###`、`####`、`#####` 生成教案表。

不要在第一行使用“同上”。第一行上方没有可合并单元格。

## 完整示例

```markdown
---
template: "jiaoan-shicao"
---

## 教学活动设计——PLC 基本指令应用

### 认识 PLC 硬件——了解 PLC 的基本组成与接线方法

#### 活动一：PLC 硬件认知

##### 0.5H

PLC 的基本组成：CPU 模块、输入模块、输出模块、电源模块。
识别 PLC 各模块的功能。

观察实训台上的 PLC 设备，识别各模块位置及功能。
完成设备模块识别记录。

展示 PLC 实物，讲解各模块的功能与作用。
点评学生识别结果。

实物展示、讲练结合

##### 0.5H

同上

根据教师要求完成模块功能说明。

同上

同上

{pagebreak}

##### 0.5H

PLC 的外部接线：输入端子接线、输出端子接线、电源接线。

按照接线图完成 PLC 的外部接线，并进行通电测试。

示范接线步骤，巡回指导学生操作，纠正接线错误。

任务驱动、示范教学

### 编写梯形图程序——掌握基本逻辑指令的使用方法

#### 活动二：基本指令编程练习

##### 1H

基本逻辑指令：LD、LDI、OUT、AND、ANI、OR、ORI。
定时器指令：TON、TOFF。
计数器指令：CTU。

使用编程软件编写“启动 - 保持 - 停止”控制程序。
完成程序的下载与调试。

讲解指令功能与梯形图编程规则。
演示编程软件的基本操作。

讲练结合、项目教学
```
