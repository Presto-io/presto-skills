# Presto Jiaoan Shicao

将课程材料、教学目标、实训任务或零散教学安排整理为 Presto `jiaoan-shicao` 模板 Markdown，生成包含封面、学习任务分析、教学活动设计、学业评价四部分的“教学设计方案（二）”，并编译为 A4 实操教案 PDF。模板契约以 `presto-official-templates/jiaoan-shicao` 2.0.0 为准：封面、学习任务分析和学业评价为纵向 A4，教学活动设计为横向 A4。

## 工作流程

```
输入材料 → 提炼课程事实 → 生成完整四部分 Markdown → 编译 PDF
```

### 步骤 1：分析输入

识别用户提供的是完整教案、课程大纲、实训任务、课堂活动草稿，还是零散素材。先提取用户明确给出的课程事实，包括课程名称、课程属性、教材名称、教学班级、计划总课时、教师姓名、使用时间、学习任务、课时、起止日期、学习目标、学习内容、学生基础、学习资源、教学活动和评价要求。

材料不完整时仍然输出最小完整骨架，不能退回为单一教学活动设计章节。补全采用保守策略：

- 可直接推断的课程主题、学习任务名、课时，可以补入封面或正文。
- 不确定的 `textbook_name`、`class_name`、`teacher_name`、`use_time` 留空。
- 不虚构学校名称、教师姓名、教学班级、教材版本、使用日期或用户没有暗示的具体评价结论。

### 步骤 2：生成封面字段

必须写入 YAML Front-Matter。即使字段为空，也保留完整字段，便于模板生成固定封面结构：

```yaml
---
template: "jiaoan-shicao"
course_name: "电工基本技能训练"
course_attribute: "基本技能课程"
textbook_name: ""
class_name: ""
total_hours: "8"
teacher_name: ""
use_time: ""
---
```

字段顺序固定为：`template`、`course_name`、`course_attribute`、`textbook_name`、`class_name`、`total_hours`、`teacher_name`、`use_time`。`course_attribute` 通常写 `基本技能课程` 或 `工学一体化课程`。

`template` 字段用于 Presto 选择模板；模板程序本身只读取封面字段。只有至少一个封面字段有值时才会生成 YAML 封面页，因此即使部分字段未知，也应把可确定的 `course_name` 或 `course_attribute` 写入 front matter。

### 步骤 3：生成学习任务分析

正文第一部分生成 `## 学习任务分析`。先写顶部字段行，再写五个 `###` 分析块：

```markdown
## 学习任务分析

学习任务：电工基本技能训练
课时：8
起止日期：

### 一、学习任务描述

### 二、学习目标

### 三、学习内容

### 四、学生情况分析

### 五、学习资源
```

五个分析块都必须保留。材料不足时用最小可用描述，不确定日期留空。

### 步骤 4：生成教学活动设计

第二部分生成 `## 教学活动设计——主题`。按 `###`、`####`、`#####` 组织活动表，不要手写普通 Markdown 表格：

| Markdown 输入 | 模板语义 |
|---|---|
| `## 教学活动设计——主题` | 进入教学活动设计章节；PDF 章节标题统一显示“教学活动设计” |
| `### 学习环节——学习单元` | 新建一张教学活动表 |
| `#### 活动名称` | “教学活动”列，模板自动加 `1.`、`2.` 编号 |
| `##### 0.5H` | “课时分配”列，标题原样显示，并生成一行活动安排 |
| `#####` 后第 1 个内容块 | “学习内容”列 |
| `#####` 后第 2 个内容块 | “学生活动”列 |
| `#####` 后第 3 个内容块 | “教师活动”列 |
| `#####` 后第 4 个内容块 | “教学方法与手段”列 |

每个 `#####` 后尽量写满四个用空行分隔的内容块。前三个内容块可以多行，模板会在单元格内自动编号为 `1. ...；`、`2. ...；`；“教学方法与手段”建议保持短语或短句。

### 步骤 5：生成学业评价

第三部分生成 `## 学业评价`。评价项使用有序列表，每行按 `考核项目；考核细则；考核方式` 三段书写，最后写 `小结：...`：

```markdown
## 学业评价

1. 安全文明操作；按安全规程完成防护用品穿戴、工位检查和通电申请；过程观察
2. 工具仪表使用；能正确选择并规范使用电工工具和万用表；实操考核
3. 导线处理质量；导线剥削、端子压接、绝缘处理和标识符合工艺要求；作品检查
4. 线路连接与检测；接线正确，检测记录完整，通电试验结果符合要求；任务成果评价
5. 协作与反思；能按小组分工完成任务并提出改进措施；学习记录评价

小结：学生能够在安全规范约束下完成本学习任务，达到基本训练要求。
```

缺少评价材料时仍保留五行评价骨架，可用课程目标和教学活动保守生成通用评价项；无法确认的具体结果不虚构。模板会至少渲染五行评价，超过五行会继续追加。

关于 YAML Front-Matter、分页、`同上` 合并单元格、多活动、多表和常见错误，按需读取 [jiaoan-shicao-format.md](references/jiaoan-shicao-format.md)。

### 步骤 6：保存文件

按用户指定位置保存；未指定时保存在当前工作目录，建议使用 `YYYYMMDD 项目名` 子文件夹。Markdown 文件名使用教案主题，去掉不适合文件名的符号。

### 步骤 7：生成 PDF

使用 stdin/stdout 管道生成 PDF，不保留中间 `.typ` 文件。

macOS 本机：

```bash
cat "教案名.md" | /Users/mrered/.presto/templates/jiaoan-shicao/presto-template-jiaoan-shicao | typst compile - "教案名.pdf"
```

OpenClaw Linux：

```bash
PRESTO=/root/.openclaw/workspace/.presto/bin/presto-template-jiaoan-shicao
TYPST=/root/.openclaw/workspace/.bin/typst
DOC=/root/.openclaw/workspace/documents

cat "$DOC/项目文件夹/文件名.md" | "$PRESTO" | "$TYPST" compile --root /root/.openclaw/workspace - "$DOC/项目文件夹/文件名.pdf"
```

PDF 文件名与 Markdown 文件同名，扩展名改为 `.pdf`。

## 触发场景

此技能在以下场景自动触发：

1. **显式调用**：用户使用 `/presto-jiaoan-shicao` 或直接点名此技能。
2. **教案生成**：用户要求生成实操教案、实训教案、职业教育教案、课堂教学活动设计。
3. **结构转换**：用户要求把课程内容、实训任务、项目化教学单元排成教案表。
4. **模板编译**：用户要求使用 Presto `jiaoan-shicao` 模板生成 Markdown 或 PDF。

## 最小示例

```markdown
---
template: "jiaoan-shicao"
course_name: "电工基本技能训练"
course_attribute: "基本技能课程"
textbook_name: ""
class_name: ""
total_hours: "8"
teacher_name: ""
use_time: ""
---

## 学习任务分析

学习任务：电工基本技能训练
课时：8
起止日期：

### 一、学习任务描述

完成常用电工工具识别、安全检查、导线处理、线路连接和检测记录。

### 二、学习目标

1. 能识别常用电工工具和低压电器元件。
2. 能按安全规范完成线路连接与检测。

### 三、学习内容

安全用电、工具仪表使用、导线处理、线路连接与检测。

### 四、学生情况分析

学生具备基础电路知识，但实操规范性需要通过示范和训练强化。

### 五、学习资源

工量具、设备：电工实训台、万用表、剥线钳、螺丝刀。
耗材：BV 导线、接线端子、灯座、开关、绝缘胶带。
其它：线路图、安全操作规程、任务书、评价量规。

## 教学活动设计——电工基本技能训练

### 安全规范与工具准备——识别常用电工工具并完成安全检查

#### 活动一：任务导入与安全规范确认

##### 0.5H

安全用电要求、个人防护用品穿戴和工位检查要点。

阅读任务书，按检查清单完成工位和工具安全确认。

说明任务标准，演示安全检查流程，提醒通电前确认要求。

任务驱动、示范教学

## 学业评价

1. 安全文明操作；按安全规程完成防护用品穿戴、工位检查和通电申请；过程观察
2. 工具仪表使用；能正确选择并规范使用电工工具和万用表；实操考核
3. 导线处理质量；导线剥削、端子压接、绝缘处理和标识符合工艺要求；作品检查
4. 线路连接与检测；接线正确，检测记录完整，通电试验结果符合要求；任务成果评价
5. 协作与反思；能按小组分工完成任务并提出改进措施；学习记录评价

小结：学生能够在安全规范约束下完成电工基本技能训练，达到基本训练要求。
```
